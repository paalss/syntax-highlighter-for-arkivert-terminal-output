# Syntax highlighter for Git Bash
![App_interface](images/app.png)

## Åpne nettside lokalt
### Installer
Clone repo og
```
npm install
```

### Serve
```
npm run start
```

Med Nodemon (for utvikling):

```
npm run dev
```
### Gå til
[localhost:5000](http://localhost:5000/)
